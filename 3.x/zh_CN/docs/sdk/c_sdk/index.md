# 3. C SDK

标签：``c-sdk`` ``区块链应用``

----------

`c-sdk`是FISCO-BCOS 3.0实现的c版本的sdk, 提供c风格的访问区块链的接口,支持rpc、amop和合约事件订阅等基础功能。用户可以用于开发c语言的区块链应用,也可以方便其他开发人员在c sdk基础上进行封装,快速开发其他语言版本的sdk。
