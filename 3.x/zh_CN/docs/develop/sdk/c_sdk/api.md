# API列表

标签：``c-sdk`` ``API``

----------

## sdk对象操作

- `bcos_sdk_create`
  - 函数原型: `void* bcos_sdk_create(struct bcos_sdk_c_config* config)`
  - 功能: 创建`sdk`对象
  - 参数:
    - config: 配置对象指针
  - 返回值:
    - 返回`sdk`对象指针
- `bcos_sdk_start`
  - 函数原型: `void bcos_sdk_start(void* sdk)`
  - 功能: 启动sdk
  - 参数:
    - sdk: `sdk`对象指针
  - 返回:
    - 无
- `bcos_sdk_stop`
  - 函数原型: `void bcos_sdk_stop(void* sdk)`
  - 功能: 停止`sdk`
    - sdk: `sdk`对象指针
  - 返回:
    - 无
- `bcos_sdk_destroy`
  - 函数原型: `void bcos_sdk_destroy(void* sdk)`
  - 功能: 停止并且销毁`sdk`对象
  - 返回值: 无
- `bcos_sdk_get_last_error`
  - 函数原型: `int bcos_sdk_get_last_error()`
  - 功能:获取上次操作的执行结果, 只针对同步操作有效
  - 返回值:
    - 0: 操作成功, 其他值,错误码
- `bcos_sdk_get_last_error_msg`
  - 函数原型: `const char* bcos_sdk_get_last_error_msg()`
  - 功能: 获取上次操作执行失败的错误信息描述
  - 返回值: 错误描述信息

## RPC接口

- `bcos_rpc_call`
  - 函数原型:
    - `void bcos_rpc_call(void* sdk, const char* group, const char* node, const char* to, const char* data,
    bcos_sdk_c_struct_response_cb callback, void* context)`
  - 功能: 调用合约，查询操作，无需区块链共识
  - 参数:
    - `sdk`: 对象指针
    - `group`: 群组ID
    - `node`: 节点名称
    - `to`: 合约地址
    - `data`: 参数编码
      - solidity合约调用时为ABI编码
      - liquid合约调用时为liquid编码
    - `callback`: 回调函数, `bcos_sdk_c_struct_response_cb`结构:

        ```c
        struct bcos_sdk_c_struct_response
        {
            int error;   // 错误码, 0: 成功, 其他表示失败
            char* desc;  // 失败时描述错误信息

            void* data;   // 返回结果指针, 成功时有效,
            size_t size;  // 返回结果大小, 成功时有效

            void* context;  // 回调上下文
        };
        ```

         **!!!注意: 回调的数据`data`仅在回调线程有效,多线程场景下,用户需自行拷贝保证数据的有效性**
    - `context`: 回调上下文, 回调时会将该参数直接返回
  - 返回值: 空
- `bcos_rpc_send_transaction`
  - 函数原型: `void bcos_rpc_send_transaction(void* sdk, const char* group, const char* node, const char* data,
    int proof, bcos_sdk_c_struct_response_cb callback, void* context)`
  - 功能: 发送交易，需要区块链共识
  - 参数:
    - `sdk`: 对象指针
    - `group`: 群组ID
    - `node`: 节点名称
    - `to`: 合约地址
    - `data`: 参数编码

      - liquid合约调用为liquid编码
    - `proof`: 回执是否返回交易回执证明，0：不返回，1：返回
    - `callback`: 参考`bcos_rpc_call`接口对`callback`的说明
    - `context`: 参考`bcos_rpc_call`接口对`context`的说明
  - 返回值: 空

  - 功能: 订阅topic
  - 参数:
    - `sdk`: 对象指针
    - `topics`: topic内容指针
    - `topic_count`: topics指针长度
- `bcos_amop_subscribe_topic_with_cb`
  - 函数原型: `void bcos_amop_subscribe_topic_with_cb(void* sdk, const char* topic, bcos_sdk_c_amop_subscribe_cb cb, void* context)`
  - 功能: 订阅topic，并设置收到topic消息的回调函数
  - 参数:
    - `sdk`: 对象指针

    - `cb`: 回调函数

- `bcos_amop_set_subscribe_topic_cb`
  - 函数原型: `void bcos_amop_set_subscribe_topic_cb(void* sdk, bcos_sdk_c_amop_subscribe_cb cb, void* context)`
  - 功能: 设置topic回调函数,接收的topic消息没有单独设置回调函数时,调用该回调函数
  - 参数:
    - `sdk`: 对象指针

  - 函数原型: `void bcos_amop_unsubscribe_topic(void* sdk, char** topics, size_t topic_count)`
  - 功能: 取消订阅topic
