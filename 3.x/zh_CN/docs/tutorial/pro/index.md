```eval_rst
.. important::
```

```eval_rst
.. note::
   包括RPC、Gateway接入层的服务和多个区块链节点Node服务组成，其中一个Node服务代表一个群组，存储采用本地RocksDB，所有Node共用接入层服务，接入层的服务可平行扩展，适用于容量可控（T级以内）的生产环境，能够支持多群组动态扩展。
```
