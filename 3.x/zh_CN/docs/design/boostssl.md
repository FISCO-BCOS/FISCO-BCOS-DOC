# 20. 公共网络组件BoostSSL

标签： ``网络组件``  ``boostssl``

----

`boostssl`是`fisco-bcos`提供的一个通用网络组件

## 目标

- 支持国密、非国密SSL连接
- 支持HTTP协议
- 支持WebSocket协议
- 简单、易用的接口
