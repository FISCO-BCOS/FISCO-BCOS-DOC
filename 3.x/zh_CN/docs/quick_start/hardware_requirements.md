## 硬件要求

```eval_rst
.. note::
    - FISCO BCOS支持x86_64和aarch64（ARM）架构的CPU
    - 由于节点多群组共享网络带宽、CPU和内存资源，因此为了保证服务的稳定性，一台机器上不推荐配置过多节点。
```

下表是单群组单节点推荐的配置，节点耗费资源与群组个数呈线性关系，您可根据实际的业务需求和机器资源，合理地配置节点数目。


|   **配置**   |   **最低配置**  |   **推荐配置**   |
| :--------: |  :--------:  | :-------: |
| 内存   |  8GB   |   32GB  |
| 核心   |  4核   |    8核    |
| 带宽   |  1Mb   |   10Mb  |

## 操作系统

- CentOS 7.2+
- Ubuntu 18.04+
- macOS 10.14+
