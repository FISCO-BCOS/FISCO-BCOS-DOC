Upgrade Guide
------------
FISCO BCOS version iteration, supports compatible upgrades between versions, supports grayscale upgrades, and during the grayscale upgrade process, the system can achieve normal consensus and block。For details, please refer to the 'Upgrade Guide'<./upgrade.html>`_ 。

FISCO BCOS design Feature control feature function is turned on and off, users can choose to turn on according to their own needs。Fix some bugs with compatibility problems through bugfix。See 'Feature and Bugfix List' for details<./feature_bugfix_list.html>`_ 。
