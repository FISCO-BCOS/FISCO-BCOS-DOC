# Development Example

Tags: "c-sdk" "Example"

The 'bcos-c-sdk / sample' directory provides some examples of sdk usage:

```shell
bcos-c-sdk/sample/
├── CMakeLists.txt
├── amop        # amop
├── eventsub    # Contract Event Subscription
└── rpc         # rpc
└── tx          # Send transaction
├── config      # Profile
```
