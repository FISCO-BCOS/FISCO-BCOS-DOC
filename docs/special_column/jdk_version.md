# JDK版本

标签：``JDK``

----

## 1. JDK版本要求
- 命令行交互控制台：
    - 控制台2.6+：JDK1.8 至 JDK14
    - 控制台1.x：大于等于1.8，推荐使用OracleJDK。
- Java SDK: JDK1.8 至 JDK14
- Web3SDK: 大于等于1.8，推荐使用OracleJDK。
- WeBASE: JDK 1.8 至JDK 14
- 区块链浏览器：JDK8或以上版本
