# 硬件要求

```eval_rst
.. note::
    由于节点多群组共享网络带宽、CPU和内存资源，因此为了保证服务的稳定性，一台机器上不推荐配置过多节点。
```

下表是单群组单节点推荐的配置，节点耗费资源与群组个数呈线性关系，您可根据实际的业务需求和机器资源，合理地配置节点数目。

```eval_rst
+----------+---------+---------------------------------------------+
| 配置     | 最低配置| 推荐配置                                    |
+==========+=========+=============================================+
| CPU      | 1.5GHz  | 2.4GHz                                      |
+----------+---------+---------------------------------------------+
| 内存     | 1GB     | 8GB                                         |
+----------+---------+---------------------------------------------+
| 核心     | 1核     | 4核                                         |
+----------+---------+---------------------------------------------+
| 带宽     | 1Mb     | 10Mb                                        |
+----------+---------+---------------------------------------------+
```

# 支持的平台

- CentOS 7.2+
- Ubuntu 16.04
- macOS 10.14+
